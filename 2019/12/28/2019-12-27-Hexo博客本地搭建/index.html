搭建Hexo本地博客及网络部署

我的第一个博客搭建与2016年，那时候才大一，也是跟着教程一起来的，三年过去了，早已经忘得不止东南西北了。考完试了，还是想把属于自己的小天地搭建起来，whatever，我的地盘我做主。

以下搭建过程是我在进行本地预览的过程，为了避免以后重新设置的时候忘记，所以记录下来了。来源

安装Node.js

这个应为我在很早以前就配置好了环境，所以没换时间。写的过程中特地去搜了一下，一步到胃，咳咳，一步到位。
1.检查是否安装过homebrew，没有安装的安装命令如下


2.安装node

检查是否安装完成npm -v、node -v
npm由于源服务器在国外下载node包经常超时，cnpm使用国内镜像，通过以下命令安装cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org
Nice done！

通过npm安装Hexo框架

1.前提是第一步已经完成了
npm install -g hexo-cli，可以用hexo -v查看版本号
2.创建一个空目录，cd到这个目录，然后要开始了！
输入hexo init初始化博客，用hexo new “BlogName”新建文章
3.生成静态文件
hexo g（g代表generate），生成博客静态文件
每次修改文章后，都需要通过hexo clean清理一下，
然后通过hexo g重新生成，
最后也不要忘了通过hexo s重新启动Hexo。
4.启动
hexo s，根据提示访问本地网页
5.更换主题
把下载的主题直接放在theme目录下，或者cd到theme目录，然后git clone也行啊。然后在_config.yml文件夹中Ctrl+F找到theme，换成你自己的主题的名字。
每次修改主题后，都需要通过hexo clean清理一下，
然后通过hexo g重新生成，
最后也不要忘了通过hexo s重新启动Hexo。
