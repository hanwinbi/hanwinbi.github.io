# 安装Node.js

这个应为我在很早以前就配置好了环境，所以没换时间。写的过程中特地去搜了一下，~一步到胃~，咳咳，一步到位。
    1.检查是否安装过homebrew，没有安装的安装命令如下
    ``` 
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" 
    ```
    2.安装node
    ```
    brew link node
    brew uninstall node//删除node，如果已经安装过会产生冲突
    brew install node
    ```
    检查是否安装完成`npm -v`、`node -v`
    npm由于源服务器在国外下载node包经常超时，cnpm使用国内镜像，通过以下命令安装cnpm
    `npm install -g cnpm --registry=https://registry.npm.taobao.org`
    Nice done！

# 通过npm安装Hexo框架

1.前提是第一步已经完成了
`npm install -g hexo-cli`，可以用`hexo -v`查看版本号
2.创建一个空目录，cd到这个目录，然后要开始了！
输入`hexo init`初始化博客，用`hexo new “BlogName”`新建文章
3.生成静态文件
`hexo g`（g代表generate），生成博客静态文件
每次修改文章后，都需要通过`hexo clean`清理一下，
然后通过`hexo g`重新生成，
最后也不要忘了通过`hexo s`重新启动Hexo。
4.启动
`hexo s`，根据提示访问本地网页
5.更换主题
把下载的主题直接放在theme目录下，或者cd到theme目录，然后`git clone`也行啊。然后在_config.yml文件夹中Ctrl+F找到theme，换成你自己的主题的名字。
每次修改主题后，都需要通过`hexo clean`清理一下，
然后通过`hexo g`重新生成，
最后也不要忘了通过`hexo s`重新启动Hexo。
6.配置到远程git
`hexo clean && hexo deploy`

### 第一步 注册 OAuth Application

首先在点击这里注册自己OAuth Application，

此处有四个内容：
```
Application name： 随意填写undefinedHomepage 
URL: 随意填写一个
Application description: 随意填写
callback URL: 填你的博客网址（例: https://user.github.io/blog/），如果有域名则填域名
```
注册完后你会得到一个 client ID 和一个 client secret，记住这两个玩意，等会hexo配置会用到

### 第二步 安装Gitalk

在你的博客下使用npm进行安装gitment
```
$ npm install --save gitalk
```

### 第三步 修改主题配置

我自己使用的是Hexo的主题，只要在主题下的_config.yml文件中找到comment即可。然后配置相关参数

```
  clientID: 'GitHub Application Client ID',
  clientSecret: 'GitHub Application Client Secret',
  repo: 'GitHub repo',
  owner: 'GitHub repo owner',
  admin: ['GitHub repo owner and collaborators, only these guys can initialize github issues'],
  id: location.pathname,      // Ensure uniqueness and length less than 50
```
github_user不知道是哪个？你在github页面中，点击你的头像看到第一行显示signed in as xxx，

就是填写这个xxx。github_repo这里是让你再新建一个仓库，用来存储评论的，不是当前的这个博客的仓库，

然后填上你仓库名的名字，对！就是单纯的名字，仓库名叫blog_comments就填blog_comments。
